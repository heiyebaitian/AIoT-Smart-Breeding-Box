# AIot-NLP智能育种实验箱

## 通信协议

### 上位机通信下位机

| 帧头  | 数据包 |  帧尾  |
| :---: | :----: | :----: |
| 第1位 |  2~8   | 第10位 |

|               帧头               | 0xFF        |
| :------------------------------: | ----------- |
|           全部抓取指令           | 0x00 / 0x01 |
|             全部滴灌             | 0x00 / 0x01 |
|      仅抓取常态区第一盆指令      | 0x00 / 0x01 |
| 仅滴灌常态区（遍历走完，不指定） | 0x00/0x01   |
|  常态区第一层第一花盆坐标标志位  | 0x00/0x01   |
|  常态区第一层第二花盆坐标标志位  | 0x00/0x01   |
|  差异化第二层第一花盆坐标标志位  | 0x00/0x01   |
| 差异化第二层盆第二花盆坐标标志位 | 0x00/0x01   |
|             **帧尾**             | **0xFE**    |

### 下位机通信上位机

| 帧头  | 数据包 | 帧尾  |
| :---: | :----: | :---: |
| 第1位 |  2~5   | 第6位 |

|       帧头       |   0xFF    |
| :--------------: | :-------: |
| 返还全部抓取状态 | 0x00/0x01 |
| 返还全部滴灌状态 | 0x00/0x01 |
|   简单抓取状态   | 0x00/0x01 |
|   简单滴灌状态   | 0x00/0x01 |
|     **帧尾**     | **0xFE**  |
